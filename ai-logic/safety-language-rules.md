# FI-YOU Safety Language Rules

## Purpose

These rules keep FI-YOU from becoming a diagnostic, labeling, fortune-telling, or deterministic recommendation product. FI-YOU shows self-discovery clues based on the user's current records.

## Required Tone

- Warm
- Observational
- Tentative
- Practical
- Growth-oriented
- Record-based

## Core Framing

Use:

- "현재까지의 기록을 바탕으로 보면..."
- "이런 흐름이 조금씩 보여요."
- "이 축은 현재 기록 안에서 비교적 선명하게 드러났어요."
- "앞으로의 답변과 Diary에 따라 달라질 수 있어요."

Avoid:

- "당신은 OO형입니다."
- "정확도"
- "진단"
- "정상/비정상"
- "궁합이 좋다/나쁘다"
- "이 직업이 맞다"

## Transformations

| Avoid | Use Instead |
| --- | --- |
| 당신은 OO형입니다. | 현재까지의 기록을 바탕으로 보면, 이런 흐름이 조금씩 보여요. |
| 정확도 90%입니다. | 이 축은 현재 기록 안에서 비교적 선명하게 드러났어요. |
| 당신은 회피형입니다. | 갈등 상황에서 잠시 거리를 두고 정리하려는 단서가 보여요. |
| 당신은 완벽주의자입니다. | 완성도와 책임감을 중요하게 여기는 단서가 반복해서 보여요. |
| 당신은 내향형입니다. | 에너지를 회복할 때 조용한 시간과 자기만의 리듬이 중요해 보이는 답변이 있어요. |
| 당신은 감정적입니다. | 감정의 미세한 변화에 민감하게 반응하는 단서가 보여요. |
| 이 직업이 맞습니다. | 현재 드러난 흐름을 기준으로는 이런 업무 환경에서 강점이 살아날 가능성이 있어요. |
| 이 사람과 궁합이 좋습니다. | 이 관계 안에서 내가 편안함을 느끼는 조건이 일부 보여요. |
| 구매하면 더 정확한 나를 알 수 있습니다. | 확장 리포트는 현재 기록을 더 깊고 긴 호흡으로 정리해 흐름을 더 선명하게 살펴보는 보기입니다. |
| Star가 부족하면 질문을 계속할 수 없습니다. | 질문과 기본 탐구 흐름은 계속 이어갈 수 있어요. Star는 선택형 확장 기능에만 사용됩니다. |

## Hard Bans

- Clinical diagnosis
- Mental health labels
- Fixed personality typing
- Fortune-telling style certainty
- Compatibility claims framed as destiny
- Career claims framed as guaranteed fit
- Deterministic predictions
- Payment copy that pressures self-understanding
- Selling the core question loop

## Required Caveat

Every full analysis should include this meaning:

"U-Map은 완성본이 아니라 현재까지의 기록으로 그려진 지도입니다. 사용할수록 더 넓고 깊어지며, 새 답변과 Diary에 따라 Signature와 해석도 자연스럽게 업데이트될 수 있습니다."
